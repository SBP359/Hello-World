# **Hello, World! - The Classic First Program**

## **Origins**
- The phrase **"Hello, World!"** was first popularized in programming by **Brian Kernighan and Dennis Ritchie** in their book *"The C Programming Language" (1978)*.
- It appeared earlier in an internal Bell Labs memo by Kernighan in **1974**.
- Used to demonstrate the basic syntax of **C**.

## **Purpose**
- Ensures that the **programming environment** is set up correctly.
- Tests whether the **compiler/interpreter** is working.
- Introduces **basic syntax** (print statements, functions, etc.).
